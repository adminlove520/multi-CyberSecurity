# 威胁狩猎 (Threat Hunting)

## 概述

威胁狩猎是通过主动搜索网络和端点数据，发现绕过现有安全控制的未知威胁。本模块覆盖狩猎方法论、基于 ATT&CK 的假设驱动狩猎、Sigma 检测规则工程、以及流量与日志异常分析。

## 关键工具

- Jupyter Notebooks — 狩猎分析环境
- Sigma — 通用检测规则格式
- MITRE ATT&CK Navigator — 覆盖度分析
- Elastic Stack / Splunk — 日志分析与狩猎
- Zeek / Suricata — 网络流量分析
