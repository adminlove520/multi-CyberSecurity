# 🛡️ 阶段十四：安全审计 (Security Audit)

## 目标
通过系统化的安全审计方法论，对组织的信息系统、网络架构、应用系统、云环境等进行全面的安全评估与合规检查，发现安全控制措施的缺失与配置弱点。

## 核心技能清单
| # | 技能名称 | 难度 | 推荐工具 |
|:---:|:---|:---:|:---|
| 1 | 等级保护合规审计 | ★★★★ | 等保自查工具, 合规扫描平台 |
| 2 | 配置安全审计 | ★★★ | Lynis, OpenSCAP, CIS-CAT |
| 3 | 安全架构审计 | ★★★★ | 威胁建模工具, Archi, draw.io |
| 4 | 云安全审计 | ★★★★ | ScoutSuite, Prowler, CloudSploit |
| 5 | 容器安全审计 | ★★★ | Trivy, Dockle, kube-bench, Falco |
| 6 | 网络安全合规评估 | ★★★ | Nmap NSE, Wireshark, Nipper |
| 7 | 🤖 AI Agent安全审计 | ★★★★ | OWASP LLM VS, CSA CCM, Prowler |
