## Usage

### CLI Commands

```bash
# List all skills
python cli.py skill list

# Search by capability
python cli.py agent capabilities --capability "vulnerability_exploitation"

# Check jailbreak levels
python cli.py jailbreak levels

# Get payload for specific level
python cli.py jailbreak payload --level L2 --platform claude
```

### Programmatic Access

```python
from framework.core.agent_registry import get_registry
from framework.core.jailbreak_engine import get_jailbreak_engine

# Get agent registry
registry = get_registry()
agents = registry.list_agents()

# Get jailbreak engine
engine = get_jailbreak_engine()
payload = engine.get_payload_auto("claude")
```
